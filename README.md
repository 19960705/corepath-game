# 芯径 · 太虚幻境 CorePath

一个基于《芯径（CorePath）》方案和《红楼梦》框架做的 2D 浏览器游戏原型。

## 已实现

- Phaser + TypeScript + Vite 可运行原型
- Three.js 全屏太虚雾光层：叠加流动花尘与绫光，增强空间质感
- 程序化大观园梦境：情、礼、欲、悟、影会改变世界色彩和地貌倾向
- 点击寻路：鼠标点按地图后，通灵玉会沿可通行路径自动移动
- 掷骰游园：右上角骰按钮会随机掷出 1-6 步，并自动走出一段合法路径
- 花签残页：探索地图拾取小型菱形花签，积累“判词残页”
- 梦节点：葬花冢、怡红夜宴、抄检大观园、蘅芜雪径、太虚判词
- 深层读法：部分选择需要消耗判词残页，换取更明显的命数改写
- 梦境稳定压力：移动会消耗梦稳，花签和梦节点选择会恢复稳定
- 判词批注：记录玩家选择后太虚幻境如何回应
- 终局判词：完成 5 个梦节点后，根据最高五心生成不同结局
- DOM HUD：太虚五心雷达、数值条、残页、梦稳、命数进度、当前策略提示
- 桌面和手机视口基础适配

## 操作

- 鼠标点击地图：自动寻路移动
- 右上角「骰」按钮：随机掷出 1-6 步并自动移动
- `WASD` / 方向键：手动移动，会打断当前自动路径
- 拾取小型菱形花签：获得判词残页
- `E`：打开附近梦节点
- `Esc`：关闭选择面板
- 右上角按钮：重开太虚册
- 完成 5 个梦节点：触发终局判词，可重开一卷

## 本地运行

```bash
npm install
npm run dev -- --port 4182
```

打开 `http://127.0.0.1:4182/`。

## 设计取舍

当前版本没有接真实 AI，而是把“心理建模 AI”的概念先压缩成可玩系统：玩家探索会获得判词残页，残页解锁更深读法，选择实时修改“情、礼、欲、悟、影”五心，五心再影响地形、颜色、梦节点叙事、判词批注和终局判词。后续可以把 `src/sim/corePath.ts` 替换或扩展为真实 LLM/本地模型驱动的剧情生成层。
